HW05 turn in set for Jake Gregg for CSE 274B

This program solves the Traveling Salesperson Problem for a set of Starbucks
locations for a specified amount of cities. It can test two forms of Graph
data structures (MatrixGraph and ListGraph) for the test which is specified in 
the testSpeedTSP function call. This will run through the TSP test until the max
timeout has been reached.

Best Times for TSP

Using a MatrixGraph: 11 cities @ 17.427 seconds, 10 cities @ 1.738 seconds

Using a ListGraph: 10 cities @ 17.328 seconds,  9 cities @ 1.668 seconds

This program meets these quality measures as specified by the HW05 measures: A, B, C, D.
